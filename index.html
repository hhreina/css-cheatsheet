Advanced CSS Selectors Cheat Sheet

Play CSS Diner to sharpen your selector skills!

Selector Example Description
Adjacent Sibling Combinator img + p Selects a paragraph immediately following an image only if both are children of the same parent element
Child Combinator ul.my-items > li Selects all list items that are direct children of the unordered list with the class "my-items"
Pseudo Selector: First Child p:first-child Selects the first paragraph in the paragraph's parent element
Pseudo Selector: Last Child p:last-child Selects the last paragraph in the paragraph's parent element
Pseudo Selector: nth Child p:nth-child{3) Selects the third paragraph in the paragraph's parent element
Pseudo Elements ::before ::after q::before {content: "«";} q::after {content: "»";} Selects all quote elements and adds quote marks before and after them. Note that content added in this way is not part of the DOM.
Pseudo Class :focus input:focus {background-color: lightcyan;} Can be used with any element, class, or id. If the forrm's input is tapped, clicked, or tabbed to from keyboard, the background changes color.
Pseudo Class :valid input:valid {border: green;} Used with forms. If the forrm's input value is valid, it gets a green border.
Pseudo Class :invalid input:invalid {border: red;} Used with forms. If the forrm's input value is invalid, it gets a red border.
